# Implementation Plans

Use this folder for project implementation plans that should be readable by both humans and agents.

Plans in this folder are tool-neutral project artifacts. They may be executed by Codex, Superpowers-style workflows, or any other agent, but the folder itself is not tied to a specific tool.

## Naming

Use date-prefixed, kebab-case filenames:

```text
YYYY-MM-DD-short-topic.md
```

Example:

```text
2026-07-04-dashboard-total-revenue-alignment.md
```

## Relationship To `docs/superpowers/plans`

Older generated plans can remain in `docs/superpowers/plans/` for history.

Going forward, put shared implementation plans in this folder when the plan is intended to be part of the project's engineering documentation rather than a tool-specific session artifact.
2026-07-04-dashboard-total-cost-net-income-alignment.md
